# assets/photos/

Shared studio photos for use in proposals (cover heroes, class cards, etc.).
Reference by URL in a proposal `CONFIG` block, e.g.
`photoUrl: "https://rfstudiosmiami.github.io/Proposals/assets/photos/aerial-yoga.jpg"`

| File | Shows | Good for |
|---|---|---|
| `aerial-yoga.jpg` | Aerial yoga class in green silks | Cover hero / aerial class card |
| `sound-bath.jpg` | Crystal singing bowls, candlelit | Sound bath / meditation class card |
| `rf-method.jpg` | Strength training with dumbbells | RF Method / strength class card |
| `bachelorette.jpg` | Group toast in hammocks (bride-to-be) | Private events / bachelorette card |

All are ~1536×1024 JPEG (q82, ~260–390 KB each), optimized from the original
~2 MB PNGs for fast page loads. The Continuum cover hero reuses `aerial-yoga.jpg`.
