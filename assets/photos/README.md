# assets/photos/

Shared studio photos for use in proposals (cover heroes, class cards, etc.).
Reference by URL in a proposal `CONFIG` block, e.g.
`photoUrl: "https://rfstudiosmiami.github.io/Proposals/assets/photos/aerial-yoga.png"`

| File | Shows | Good for |
|---|---|---|
| `aerial-yoga.png` | Aerial yoga class in green silks | Cover hero / aerial class card |
| `sound-bath.png` | Crystal singing bowls, candlelit | Sound bath / meditation class card |
| `rf-method.png` | Strength training with dumbbells | RF Method / strength class card |
| `bachelorette.png` | Group toast in hammocks (bride-to-be) | Private events / bachelorette card |

All are ~1536×1024 PNG. For lighter pages, consider re-encoding to `.webp`/`.jpg`.
