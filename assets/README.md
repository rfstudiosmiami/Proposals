# assets/

Shared images for proposals — logos, hero photos, background images.

Upload files here, then reference them in a proposal's `CONFIG` block by URL:

```
https://rfstudiosmiami.github.io/Proposals/assets/your-file.png
```

## Conventions

- Lowercase, hyphenated filenames: `rfstudios-logo.png`, `continuum-hero.jpg`
- Prefer optimized web formats (`.webp`/`.jpg` for photos, `.png`/`.svg` for logos)
- Keep large hero images here rather than base64-embedding them in the HTML
