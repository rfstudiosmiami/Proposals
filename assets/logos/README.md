# assets/logos/

RF Studios brand logos. Reference in a proposal `CONFIG` block by URL, e.g.
`rfStudiosLogoUrl: "https://rfstudiosmiami.github.io/Proposals/assets/logos/rf-studio-logo.svg"`

| File | Notes |
|---|---|
| `rf-studio-logo.svg` | Primary RF Studio wordmark (SVG) |
| `rf-studio-logo-01.svg` | Wordmark variant 01 |
| `rf-studios-logo-02.svg` | Wordmark variant 02 |
| `rf-symbol.svg` | RF symbol / icon mark only |
| `rf-hero-logo-tagline-center.svg` | Hero lockup with tagline, centered (SVG — prefer this) |
| `rf-hero-logo-tagline-center.png` | Same hero lockup as PNG (1189×721, for non-SVG contexts) |
| `rf-windows-bounce-fly-float-repeat.svg` | Animated "Bounce, Fly, Float, Repeat" mark |

Prefer SVG wherever possible — sharp at any size, tiny file. Use the PNG only where SVG isn't supported.
